# maintenance

Простая и элегантная страница для отображения во время проведения технических работ на сайте.

## фаст старт

1. Загрузите файлы `maintenance.html` и `.htaccess` в корневую директорию вашего сайта
2. Настройте время окончания работ в файле `maintenance.html`
3. При необходимости добавьте свой IP-адрес в `.htaccess` для доступа к сайту

## настройки времени окончания работ

Откройте файл `maintenance.html` и найдите в JavaScript следующую строку:

```javascript
const endTime = new Date('2025-12-20T15:30:00').getTime();
```

Замените дату и время на нужные вам. Формат: `YYYY-MM-DDTHH:MM:SS`

Примеры:
- `2025-01-15T18:00:00` - 15 января 2025 года, 18:00
- `2025-02-28T23:30:00` - 28 февраля 2025 года, 23:30
- `2025-12-31T12:00:00` - 31 декабря 2025 года, 12:00

## настройка .htaccess

### еще инфо

Файл `.htaccess` автоматически перенаправляет всех посетителей на страницу `maintenance.html`. При этом:
- Ваш основной сайт остается нетронутым
- Статические файлы (CSS, JS, изображения) доступны
- Поисковые системы видят временный редирект (302)

### добавление вашего IP-адреса

Чтобы администраторы могли видеть основной сайт во время технических работ, добавьте их IP-адреса в `.htaccess`.

Откройте файл `.htaccess` и добавьте строку ПЕРЕД строкой `RewriteRule`:
